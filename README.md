# Input Latency 35

Input Latency 35 is a Python-based tool designed to measure and analyze the input latency in gaming setups. By providing precise measurements and detailed reports, developers and gamers can optimize their systems for improved responsiveness and performance.

## Features

- **Real-Time Input Measurement**: Capture input latency in real-time, allowing for immediate feedback on system performance.
- **Detailed Analytics**: Generate comprehensive reports that break down latency data into actionable insights, making it easier to identify bottlenecks.
- **Customizable Testing Parameters**: User-defined settings for inputs, timing thresholds, and testing environment to suit individual needs.
- **Cross-Platform Compatibility**: Works seamlessly across major operating systems (Windows, macOS, Linux), ensuring versatility for all gamers.

## Installation

To install Input Latency 35, follow these steps:

1. Ensure you have Python 3.7 or higher installed. You can download it from the [official Python website](https://www.python.org/downloads/).
2. Clone this repository:

   ```bash
   git clone https://github.com/Developer/input-latency-35.git
   ```
   
3. Navigate into the cloned directory:

   ```bash
   cd input-latency-35
   ```
   
4. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Basic Usage

To start measuring input latency, run the following command:

```bash
python input_latency.py
```

Once the program is running, follow the on-screen prompts to initiate tests. Detailed results will be displayed after each test, and a summary report can be generated by using:

```bash
python input_latency.py --report
```
