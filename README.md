# Input Latency 35

Input Latency 35 is a Python-based tool designed to measure and analyze input latency in gaming environments. By accurately tracking input delay, developers can enhance gameplay experiences and ensure smoother interactions for players.

## Features

- **Real-Time Latency Measurement**: Capture and display input latency metrics in real-time, allowing for immediate adjustments and optimizations.
- **Customizable Input Sources**: Configure various input devices, such as keyboards, mice, or controllers, to suit different testing scenarios.
- **Detailed Performance Analytics**: Generate comprehensive reports that break down input delay across multiple game sessions, providing insights for improvement.
- **Cross-Platform Compatibility**: Built to work seamlessly on Windows, macOS, and Linux, ensuring broad accessibility for game developers.

## Installation

To get started with Input Latency 35, clone the repository and install the required dependencies:

```bash
git clone https://github.com/Developer/input-latency-35.git
cd input-latency-35
pip install -r requirements.txt
```

Ensure you have Python 3.7 or higher installed on your system.

## Basic Usage Example

Once the installation is complete, run the following command to start measuring input latency:

```bash
python measure_latency.py --input-device keyboard
```

In this example, replace `keyboard` with your preferred input device. The program will display latency metrics in real-time, enabling you to identify and troubleshoot potential performance issues.
